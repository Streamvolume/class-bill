# 班费公示平台

> 药学2023级231班 · 班费管理与公示系统

一个面向班级内部的轻量班费公示平台，数据公开透明，账目可追溯，由班委维护。

---

## 功能

**公示页**

- 入学至今天数、出账次数、账户余额、人均剩余四项实时统计
- 完整收支明细表，支持按入账 / 出账 / 待确认筛选
- 每次刷新随机切换配色主题
- 右下角 Live2D 互动形象，支持视线跟随，随机对话

**管理后台**（班委专用）

- 凭动账密码登录，GitHub Token 经 AES-256-GCM 本地加密，不以明文存储
- 新增、编辑、删除账单条目
- 主管理员核实确认机制，未核实条目公示页有标记
- 账单变更直接写入 GitHub 仓库，每次操作自动形成 commit 记录

---

## 数据与隐私

- 账单数据存于本仓库 `bills.json`，公开可查
- 集体缴费仅记人数与总额，不记录个人信息
- 出账仅记经办人角色（如生活委员），不记录真实姓名

---

## 页面入口

| 页面 | 说明 |
|---|---|

| `admin-login.html` | 班委登录页 |
